Featured Projects

Production-focused systems across Agentic AI, RAG, and Deep Learning

FinPilot-AI

Agentic GenAI Finance Copilot

Problem

Financial planning tools either rely on static calculators or let LLMs hallucinate numerical outputs — creating unreliable results for critical financial decisions.

Architecture

Tool-calling agentic pipeline where the LLM acts as an orchestrator — it reasons and explains, but never computes. Dedicated Python tools handle EMI calculation, stress testing, and affordability analysis with deterministic accuracy.

Key Engineering Decisions

Separated computation from generation to eliminate hallucination risk. Added XAI panel so users see why recommendations are made, not just what they are.

Groq LLM Agentic AI Streamlit XAI

Fashion Product Intelligence

Deep Learning Recommendation System

Problem

E-commerce platforms need automated product categorization and visual similarity search — manual tagging doesn't scale across 44K+ product images.

Architecture

ResNet50 transfer learning pipeline for image classification, with feature extraction layer producing embeddings. Cosine similarity engine matches query images against the embedding space for real-time recommendations.

Key Engineering Decisions

Used transfer learning over training from scratch — achieved production-quality accuracy with limited compute. Precomputed embeddings for O(1) similarity lookup at inference time.

TensorFlow ResNet50 Transfer Learning Streamlit

GitHub RAG Chatbot

Fully Local Code Q&A System

Problem

Understanding unfamiliar codebases requires reading thousands of lines. Existing tools send proprietary code to cloud APIs — unacceptable for private repositories.

Architecture

Ingests any GitHub repository, chunks files by semantic boundaries, builds a vector index with LlamaIndex, and answers code-level questions via Ollama — entirely local.

Key Engineering Decisions

Zero cloud API dependency — all inference runs locally via Ollama. HuggingFace embeddings for code-aware vectorization. Designed for air-gapped environments.

LlamaIndex RAG Ollama HuggingFace

News Popularity Intelligence

Transformer-Based NLP Ranking System

Problem

Editors need to prioritize articles before engagement data exists — the cold-start problem in content ranking.

Architecture

DistilBERT embeddings combined with engineered signals (urgency, sentiment, readability) fed into a ranking model with weak supervision. Produces explainable popularity scores for editorial decision support.

Key Engineering Decisions

Used weak supervision to bootstrap labels without manual annotation. Multi-signal fusion approach outperforms pure embedding-based ranking on cold-start content.

AI Skill Gap Analyzer

GenAI-Powered Adaptive Assessment Engine

Problem

Learners struggle to identify specific skill gaps against target job roles — generic skill assessments don't adapt to individual knowledge levels.

Architecture

Groq LLM generates scenario-based questions tailored to target roles, evaluates responses in real-time, maps skill gaps, and produces personalized upskilling roadmaps — deployed live on Streamlit Cloud.

App User Behavior Segmentation

Unsupervised ML Analytics

Problem

Product teams need to identify distinct user behavior groups to drive targeted engagement strategies — one-size-fits-all approaches miss high-value users.

Architecture

Unsupervised pipeline: feature engineering from app usage data → StandardScaler normalization → PCA dimensionality reduction → K-Means clustering with Elbow/Silhouette optimization → segment profiling dashboard.

Key Engineering Decisions

Applied PCA before clustering to reduce noise and improve cluster separation. Elbow method + silhouette scores for optimal K selection. Visualized clusters in 2D/3D for stakeholder communication.
